Responsive Typography Example

Subheading Adapts to Screen Size

This paragraph demonstrates responsive typography. On mobile devices (< 768px), the text is smaller for readability. On tablets (768px - 1024px), it increases to a medium size. On desktop screens (> 1024px), the text becomes larger for comfortable reading from a distance.

The font sizes automatically adjust based on the viewport width using CSS media queries. This ensures optimal readability across all devices. Try resizing your browser window to see how the text scales appropriately for different screen sizes.

Breakpoints Used:
Mobile: < 768px (14px base font)
Tablet: 768px - 1024px (16px base font)
Desktop: > 1024px (18px base font)
